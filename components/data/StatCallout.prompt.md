Big mono figure block used across PRNG internal management reports — label, figure, sub-note. Color the figure by financial meaning.

```jsx
<StatCallout label="Operating Loss" value="-$331,046.92" sub="12 months" tone="negative" />
<StatCallout label="Gross Margin" value="80.7%" tone="positive" />
```

Tones: `negative` (red), `positive` (green), `warning` (amber), `accent` (pink), `neutral` (plum). Lay several in a grid for a report header.
