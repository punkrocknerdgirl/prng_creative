Sharp-cornered, uppercase mono action button — use for any primary/secondary call to action. Pink is the brand primary; use `ghost` on dark plum panels.

```jsx
<Button variant="primary" size="lg" as="a" href="#">Book a Free Call</Button>
<Button variant="ghost">See What I Do →</Button>
```

Variants: `primary` (pink fill), `secondary` (plum fill), `ghost` (outline for dark backgrounds), `quiet` (light outline). Sizes: `sm` / `md` / `lg`. Renders as any tag via `as` (use `as="a"` for links).
