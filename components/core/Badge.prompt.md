Pill-shaped mono micro-label for credentials/qualifiers — the only rounded shape in a sharp system. Outline style, never filled.

```jsx
<Badge tone="pink">QBO ProAdvisor Level 2</Badge>
<Badge tone="muted">Texas-Based</Badge>
```

Tones: `pink` (on dark heroes), `plum` / `muted` (on light). For a filled square chip use `Pill` instead.
