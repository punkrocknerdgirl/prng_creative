The workhorse surface: white, 4px radius, soft plum shadow, pink accent bar. `accent="top"` for service/feature grids, `accent="left"` for contact/info rows.

```jsx
<Card accent="top">
  <h3>Bookkeeping</h3>
  <p>Full-cycle bookkeeping, month-end close, and reporting.</p>
</Card>
```

Set `hover={false}` for static cards. Compose `Pill` rows, `Eyebrow`, and `Button` inside.
