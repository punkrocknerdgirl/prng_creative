# PRNG Bookkeeping — Marketing Site UI Kit

High-fidelity recreation of **prngbooks.com**, the client-facing landing page for PRNG Bookkeeping Services. Source: `github.com/punkrocknerdgirl/prngbooks`.

## Screens
- `Nav.jsx` — fixed dark nav, pink underline, uppercase links + Book-a-Call CTA.
- `Hero.jsx` — dark split hero: logo, "Bookkeeper | Systems Architect", "Professional Chaos Untangler", tagline, badges, photo with plum gradient wash.
- `Services.jsx` — 6-card service grid (emoji icons + `Pill` rows).
- `Sections.jsx` — About (framed cartoon + creds), Work-With-Me (numbered steps + CTA box on plum), Contact cards, footer.

## Composes
`Button`, `Badge`, `Pill`, `Card`, `Eyebrow` from the design system bundle (`window.PRNGDesignSystem_4a3ea3`).

## Voice
Confident, plain-spoken, anti-corporate. "Clean books. Clear systems. No BS." Emoji are used intentionally as service-card icons.
